## Contexto da Aplicação

O TaskLog é um aplicativo desenvolvido para auxiliar no gerenciamento e acompanhamento de atividades relacionadas a projetos de pesquisa e extensão. A aplicação permite cadastrar tarefas, registrar descrições das atividades a serem realizadas e acompanhar seu status de execução.

Quando uma atividade é concluída, ela é movida para a lista de atividades realizadas, facilitando a organização do cronograma e o acompanhamento do progresso do projeto. Além disso, o aplicativo prevê o uso da câmera do dispositivo para registrar evidências fotográficas das atividades executadas.

O projeto foi desenvolvido na disciplina de Programação para Dispositivos Móveis 1 utilizando React Native com Expo, SQLite para persistência local de dados e recursos de hardware do dispositivo móvel.
